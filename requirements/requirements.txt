# System Requirements for Intel Unnati Project

## Hardware Requirements
- Intel Iris Xe Graphics (TigerLake GT2) or compatible Intel GPU
- 8GB+ RAM
- Ubuntu 22.04 LTS or compatible Linux distribution

## Software Requirements
- Docker Engine 20.10+
- X11 server (for visual output)
- Intel GPU drivers
- OpenVINO Runtime 2023.0+

## Docker Base Image
intel/dlstreamer:latest

## Models Required
- person-vehicle-bike-detection-crossroad-0078 (OpenVINO Model Zoo)
- person-attributes-recognition-crossroad-0230 (OpenVINO Model Zoo)

## Performance Monitoring Tools
- intel_gpu_top (Intel GPU Tools)
- System resource monitoring capabilities
